# AI Impact on Jobs by 2030

## Project Overview

This project analyzes the **impact of Artificial Intelligence on jobs by the year 2030** using Machine Learning techniques.
The goal is to explore how automation and AI technologies may affect different job sectors and employment trends.

Using data analysis and machine learning models, the project predicts job transformation trends and visualizes patterns in the dataset.

---

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## Machine Learning Models Used

The following algorithms were implemented to analyze and predict job impact trends:

* Decision Tree
* K-Means Clustering
* Linear Regression
* Random Forest

---

## Project Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Data visualization using Matplotlib and Seaborn
* Machine learning model training and evaluation
* Insights into future job trends influenced by AI

---

## Visualizations

The project uses **Matplotlib and Seaborn** to create visualizations such as:

* Job trend distributions
* Sector impact comparisons
* Model prediction insights

---

## Project Goal

The main objective of this project is to understand:

* Which industries are most affected by AI
* The future demand for certain skills
* How machine learning can help analyze job transformation patterns
